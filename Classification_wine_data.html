Wine data¶

The data retrieved from (https://archive.ics.uci.edu/ml/datasets/wine) is the results of a chemical analysis of wines grown in the same region in Italy by three different cultivators. There are 13 different measurements taken for different constituents found in the three types of wine.

Part2. PCA Data¶

Data Preprocessing¶

  • With dimensionality reduction, we retrieve more important variables from original data.

    Then, we check if the classification would be improved

  • Retain 5 variables which account for more than 80% variance of original data
  • Train test split -> Standization -> PCA
In [ ]:
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.30, random_state = 410878048)
# Standardize the test data by means and std from train data
X_var_mean = np.array(np.mean(X_train, axis = 0)).reshape(1, -1)
X_var_std = np.array(np.std(X_train, axis = 0)).reshape(1, -1)
X_test_std = (X_test - X_var_mean)/ X_var_std

# Standardize the train data
scaler = StandardScaler()
X_train_std = scaler.fit_transform(X_train)

pca = PCA()
pca.fit(X_train_std)
# decide the retained number of PCs
cumulative_variances = np.cumsum(pca.explained_variance_)
percentage = [x/X.shape[1] * 100 for x in cumulative_variances]
number_retained = list(np.array(percentage) > 80).count(False) + 1
explained_variance = pca.explained_variance_[:number_retained]
tick_of_pc = [x for x in range(1, len(explained_variance)+1)]

X_train = pca.transform(X_train_std)[:, :number_retained]
X_test = pca.transform(X_test_std)[:, :number_retained]

# show pca result
## plot bar chart
color1 = "blue"
plt.bar(tick_of_pc, explained_variance, color=color1, alpha=0.5)
plt.xlabel('The Number of Principal Components')
plt.ylabel('Explained Variance', color=color1)
plt.tick_params(axis='y', color=color1)
## create secondary y-axis
ax3 = plt.twinx()
## plot cumulative percentage line chart
ax3.plot(tick_of_pc, percentage[:number_retained], color='purple', marker='o', alpha = 0.5)
ax3.set_ylabel('Cumulative Percentage (%)', color='purple')
ax3.tick_params(axis='y', labelcolor='purple')

Classification Result¶

  • This time we try not to tune any parameters.
  • In this cases, logistic regression model performs better than SVM, Neural Network model.
  • However, the pca data does not appear to better than the original data, which means dimensonality reduction might not be neccessary in this small data.

dataset = [pred_LogReg, pred_SVM, pred_NN]
markers = ["o", "v", "s"]
group_distinct = np.unique(y_test)
# 'coolwarm' colormap
coolwarm_colors = plt.get_cmap('viridis')(np.linspace(0, 1, len(group_distinct)))
label_colors = {label: color for label, color in zip(group_distinct, coolwarm_colors)}

# Create a figure and a set of subplots
fig, axs = plt.subplots(1, 3, figsize=(19, 5))
for j, pred_data in enumerate(dataset):
    color_array = np.array([label_colors[x] for x in y_test])
    # label red if misclassification
    color_array[y_test != pred_data] = [1, 0, 0, 1]
    for i in range(len(group_distinct)):
        axs[j].scatter(X_test[pred_data == group_distinct[i], 0], X_test[pred_data == group_distinct[i], 1], c = color_array[pred_data == group_distinct[i]]\
                    , marker = markers[i], alpha=0.5, label = group_distinct[i], s= 80)
    axs[j].legend()

Conclusion¶

  • It is feasible to use three models to classify observations from the evidence more than half of test data correctly identified.
  • The logistic regression preforms the best no matter which data uses. But, it just performed slightly better than other models.
  • Dimensionality reduction in this case reduces the variables from 13 to 5. It not only loses data information but also gets worse result. It proves that it is not necessary to perform this for every dataset.
